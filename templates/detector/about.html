About the VerifAI project

A deployable deepfake detection platform for image and video analysis, presented through a professional Django web application.

Project summary

VerifAI is an AI-based deepfake detection system designed to classify uploaded media as real or fake. The final project focuses on image and video detection and intentionally excludes audio analysis to preserve stability.

Technology stack

Django TensorFlow MobileNetV2 OpenCV K-Means SQLite

System workflow

Image input -> preprocess -> CNN prediction -> Real or Fake
Video input -> frame extraction -> embeddings -> K-Means keyframes -> majority vote
Login -> upload -> analyze -> save report -> review history

Why Django

Django provides authentication, routing, template rendering, and database support. It gives the project a deployable product layer and satisfies the minimum DBMS requirement through login and stored user reports.

Model and detection logic

The model is built in TensorFlow using MobileNetV2 transfer learning for binary classification. Image uploads are analyzed directly, while videos are processed through frame extraction and K-Means clustering to reduce redundant frames and evaluate representative keyframes.

Limitation

The current system focuses on spatial visual patterns and does not model temporal inconsistencies across frames. It is intended as a stable academic project and demo-ready deepfake screening system.
